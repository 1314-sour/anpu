# 用户中心功能实现说明

## ✅ 已完成内容

### 数据库表
1. **user_profiles** - 用户资料扩展表
2. **user_settings** - 用户系统设置表
3. **operation_logs** - 操作日志表
4. **user_feedbacks** - 意见反馈表

### 后端API
所有接口需要携带token认证,路径前缀: `/api/v1/user`

#### 1. 基本资料
- `GET /profile` - 获取用户资料
- `PUT /profile` - 更新用户资料

#### 2. 安全设置
- `POST /security/password` - 修改密码
- `POST /security/avatar` - 更新头像

#### 3. 系统设置
- `GET /settings` - 获取系统设置
- `PUT /settings` - 更新系统设置

#### 4. 操作日志
- `GET /logs` - 获取操作日志(支持分页和筛选)

#### 5. 意见反馈
- `POST /feedback` - 提交反馈
- `GET /feedback` - 获取反馈列表

### 前端API封装
文件: `src/api/user.js`
- getUserProfile()

## 🔧 前端页面集成示例

### 1. Profile.vue (基本资料)

```javascript
import { getUserProfile, updateUserProfile } from '@/api/user'

export default {
  data() {
    return {
      form: {
        username: '',
        phone: '',
        email: '',
        contactPerson: '',
        address: ''
      }
    }
  },
  mounted() {
    this.loadProfile()
  },
  methods: {
    async loadProfile() {
      try {
        const res = await getUserProfile()
        this.form = {
          username: res.data.username,
          phone: res.data.phone,
          email: res.data.email,
          contactPerson: res.data.contact_person,
          address: res.data.address
        }
      } catch (error) {
        console.error('加载失败:', error)
      }
    },
    async onSubmit() {
      try {
        await updateUserProfile({
          contact_person: this.form.contactPerson,
          address: this.form.address
        })
        this.$message.success('保存成功')
      } catch (error) {
        console.error('保存失败:', error)
      }
    }
  }
}
```

### 2. Security.vue (安全设置)

```javascript
import { getUserProfile, changePassword, updateAvatar } from '@/api/user'

export default {
  data() {
    return {
      userInfo: {
        username: '',
        phone: '',
        email: '',
        avatar: '',
        registerTime: '',
        securityQuestionSet: false
      }
    }
  },
  mounted() {
    this.loadUserInfo()
  },
  methods: {
    async loadUserInfo() {
      try {
        const res = await getUserProfile()
        this.userInfo = {
          username: res.data.username,
          phone: res.data.phone,
          email: res.data.email,
          avatar: res.data.avatar,
          registerTime: res.data.register_time,
          securityQuestionSet: res.data.security_question_set
        }
      } catch (error) {
        console.error('加载失败:', error)
      }
    },
    async handleAvatarSuccess(res, file) {
      const imageUrl = URL.createObjectURL(file.raw)
      try {
        await updateAvatar({ avatar_url: imageUrl })
        this.userInfo.avatar = imageUrl
        this.$message.success('头像更新成功')
      } catch (error) {
        console.error('更新失败:', error)
      }
    }
  }
}
```

### 3. System.vue (系统设置)

```javascript
import { getUserSettings, updateUserSettings } from '@/api/user'

export default {
  data() {
    return {
      messageTypes: [],
      notifyTypes: [],
      layoutType: '',
      kgDisplay: ''
    }
  },
  mounted() {
    this.loadSettings()
  },
  methods: {
    async loadSettings() {
      try {
        const res = await getUserSettings()
        this.messageTypes = res.data.message_types
        this.notifyTypes = res.data.notify_types
        this.layoutType = res.data.layout_type
        this.kgDisplay = res.data.kg_display
      } catch (error) {
        console.error('加载失败:', error)
      }
    },
    async saveSettings() {
      try {
        await updateUserSettings({
          message_types: this.messageTypes,
          notify_types: this.notifyTypes,
          layout_type: this.layoutType,
          kg_display: this.kgDisplay
        })
        this.$message.success('设置已保存')
      } catch (error) {
        console.error('保存失败:', error)
      }
    }
  }
}
```

### 4. Logs.vue (操作日志)

```javascript
import { getOperationLogs } from '@/api/user'

export default {
  data() {
    return {
      filters: {
        object: '',
        account: '',
        type: '',
        dateRange: []
      },
      logs: [],
      total: 0,
      page: 1,
      pageSize: 10
    }
  },
  mounted() {
    this.loadLogs()
  },
  methods: {
    async loadLogs() {
      try {
        const res = await getOperationLogs({
          object: this.filters.object,
          account: this.filters.account,
          type: this.filters.type,
          start_date: this.filters.dateRange[0],
          end_date: this.filters.dateRange[1],
          page: this.page,
          page_size: this.pageSize
        })
        this.logs = res.data.list
        this.total = res.data.total
      } catch (error) {
        console.error('加载失败:', error)
      }
    }
  }
}
```

### 5. Feedback.vue (意见反馈)

```javascript
import { submitFeedback } from '@/api/user'

export default {
  data() {
    return {
      form: {
        type: 'function',
        content: ''
      }
    }
  },
  methods: {
    async onSubmit() {
      if (!this.form.content) {
        this.$message.warning('请输入建议内容')
        return
      }
      
      try {
        await submitFeedback({
          type: this.form.type,
          content: this.form.content
        })
        this.$message.success('反馈提交成功')
        this.form.content = ''
      } catch (error) {
        console.error('提交失败:', error)
      }
    }
  }
}
```

## 📊 数据库表结构

### 1. user_profiles (用户资料)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| user_id | INT | 用户ID |
| contact_person | VARCHAR(50) | 联系人 |
| address | VARCHAR(255) | 详细地址 |
| phone | VARCHAR(20) | 手机号 |
| security_question_set | BOOLEAN | 是否设置密保 |

### 2. user_settings (系统设置)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| user_id | INT | 用户ID |
| message_types | JSON | 消息类型 |
| notify_types | JSON | 提醒方式 |
| layout_type | VARCHAR(20) | 排版方式 |
| kg_display | VARCHAR(20) | KG展示方式 |

### 3. operation_logs (操作日志)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| user_id | INT | 用户ID |
| account | VARCHAR(50) | 操作账号 |
| object | VARCHAR(100) | 操作对象 |
| type | VARCHAR(50) | 操作类型 |
| content | TEXT | 操作内容 |
| ip_address | VARCHAR(50) | IP地址 |
| created_at | TIMESTAMP | 操作时间 |

### 4. user_feedbacks (意见反馈)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| user_id | INT | 用户ID |
| type | VARCHAR(20) | 反馈类型 |
| content | TEXT | 反馈内容 |
| status | VARCHAR(20) | 处理状态 |
| reply | TEXT | 回复内容 |
