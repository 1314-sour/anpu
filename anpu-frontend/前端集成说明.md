# 前端API集成说明

## ✅ 已完成的页面集成

### 1. 基本资料 (Profile.vue)

**集成功能:**
- ✅ 页面加载时自动获取用户资料
- ✅ 提交时更新联系人和地址信息
- ✅ 显示加载状态和错误提示

**使用的API:**
- `getUserProfile()` - 获取用户资料
- `updateUserProfile(data)` - 更新用户资料

**数据映射:**
```javascript
// 后端字段 → 前端字段
contact_person → contactPerson
address → address
username → username (只读)
phone → phone (只读)
email → email (只读)
```

---

### 2. 安全设置 (Security.vue)

**集成功能:**
- ✅ 页面加载时自动获取用户信息
- ✅ 根据绑定情况动态计算安全等级
- ✅ 显示注册时间、手机号、邮箱、密保状态

**使用的API:**
- `getUserProfile()` - 获取用户安全信息

**数据映射:**
```javascript
// 后端字段 → 前端字段
username → username
register_time → registerTime
avatar → avatar
phone → phone
email → email
security_question_set → securityQuestionSet
```

**安全等级计算:**
- 基础分: 25 (密码已设置)
- 手机号: +25
- 邮箱: +25
- 密保问题: +25
- 总分 < 60: 低 (红色)
- 总分 60-80: 中 (橙色)
- 总分 >= 80: 高 (绿色)

### 3. 系统设置 (System.vue)

**集成功能:**
- ✅ 页面加载时自动获取系统设置
- ✅ 提交时保存所有设置项
- ✅ 显示加载和保存状态

**使用的API:**
- `getUserSettings()` - 获取系统设置
- `updateUserSettings(data)` - 更新系统设置

**数据映射:**
```javascript
// 后端字段 → 前端字段
message_types → messageTypes (数组)
notify_types → notifyTypes (数组)
layout_type → layoutType (字符串)
kg_display → kgDisplay (字符串)
```

**可选值:**
- 消息类型: ['保留消息', '工单消息', '到期提醒', '系统公告']
- 提醒方式: ['弹窗提醒', '浏览器标签闪烁', '声音提醒']
- 页面排版: '原始比例' | '完全自适应'
- KG展示: '列表展示' | '分组展示'

### 4. 操作日志 (Logs.vue)

**集成功能:**
- ✅ 页面加载时自动获取日志列表
- ✅ 支持按对象、账号、类型筛选
- ✅ 支持按日期范围筛选
- ✅ 分页显示,每页10条
- ✅ 显示加载状态

**使用的API:**
- `getOperationLogs(params)` - 获取操作日志

**查询参数:**
```javascript
{
  object: '操作对象',     // 可选
  account: '操作账号',    // 可选
  type: '操作类型',       // 可选
  start_date: '开始日期', // 可选,格式: YYYY-MM-DD
  end_date: '结束日期',   // 可选,格式: YYYY-MM-DD
  page: 1,                // 当前页码
  page_size: 10           // 每页数量
}
```

**返回数据:**
```javascript
{
  list: [
    {
      account: '操作账号',
      object: '操作对象',
      type: '操作类型',
      content: '操作内容',
      created_at: '2025-12-05 14:30:00'
    }
  ],
  total: 50  // 总记录数
}
```

## 🔑 认证说明

所有API调用都会自动携带JWT token(在`src/utils/request.js`中配置):

```javascript
// 请求拦截器自动添加token
config.headers['Authorization'] = `Bearer ${getToken()}`
```

如果token过期,会返回401错误,需要重新登录。

## 📝 使用示例

### 示例1: 基本资料页面

```vue
<script>
import { getUserProfile, updateUserProfile } from '@/api/user'

export default {
  data() {
    return {
      form: { contactPerson: '', address: '' }
    }
  },
  mounted() {
    this.loadProfile()
  },
  methods: {
    async loadProfile() {
      const res = await getUserProfile()
      this.form = res.data
    },
    async onSubmit() {
      await updateUserProfile(this.form)
      this.$message.success('保存成功')
    }
  }
}
</script>
```

### 示例2: 操作日志查询

```vue
<script>
import { getOperationLogs } from '@/api/user'

export default {
  data() {
    return {
      logs: [],
      page: 1,
      pageSize: 10
    }
  },
  methods: {
    async loadLogs() {
      const res = await getOperationLogs({
        type: '登录账号',
        page: this.page,
        page_size: this.pageSize
      })
      this.logs = res.data.list
    }
  }
}
</script>
```
